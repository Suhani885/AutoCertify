### Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/Suhani885/AutoCertify.git
   cd AutoCertify
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
